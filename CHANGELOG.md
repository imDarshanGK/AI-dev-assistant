## [1.0.0] - 2026-01-01

### Added
- Initial release of QyverixAI
- Rule-based code analysis engine covering 40+ patterns
- Support for Python, JavaScript, TypeScript, Java, C++, PHP, and Rust
- AST-based Python analysis for deep bug detection
- Multi-platform code suggestion system
- Streaming API endpoint for real-time analysis
- In-memory and Redis cache support
- Share endpoint for saving and retrieving analysis results
